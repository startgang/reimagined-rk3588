# Тюнинг и настройка профилей камер

Для получения качественного изображения необходим этап настройки (тюнинга) параметров ISP:
- Подготовка: Установка на ПК с Windows инструмента RKISP_Tuner_v2.x, а на целевое устройство — сервиса rkaiq_tool_server.
- Сбор RAW-данных: Использование встроенной в Tuner функции Capture Tool для записывания «сырых» кадров с сенсора.
- Калибровка и настройка: В Tuner проводится калибровка сенсора и настройка модулей ISP: баланс белого (AWB), автоэкспозиция (AE), шумоподавление (3DNR), цветокоррекция (CCM) и т.д.
- Экспорт и интеграция: Финальный IQ-файл (.json/.xml) загружается в систему. Профиль сенсора IMX415 может занимать ~120-150 кБ. Путь к файлу прописывается в Device Tree.
